# Basic Browser Tracker

This is an easy project for beginners.

You can use this repository to:
- Track daily learning
- Write small notes
- Practice editing files
- Build a habit of daily GitHub commits

How to use:
1. Open any file
2. Add a small change
3. Save the file
4. Commit your update

Goal:
Make small progress every day.

Notes:
Write your daily tracking notes below.
